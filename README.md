# 📚 CRUD de Livros com Java + Firebase

## 🚀 Sobre o projeto

Este projeto consiste em um sistema CRUD (Create, Read, Update, Delete) desenvolvido em Java, com integração ao Firebase para persistência de dados em tempo real.

A aplicação permite o gerenciamento de livros, incluindo cadastro, listagem, edição e exclusão de registros.

## 🎓 Contexto acadêmico

Este projeto foi desenvolvido durante as aulas da faculdade, com orientação do professor Mauricio Asenjo, como forma de aplicar na prática conceitos de desenvolvimento de sistemas e integração com banco de dados em nuvem.

## 🧠 Objetivo

O projeto foi desenvolvido com o objetivo de praticar:

* Programação Orientada a Objetos (POO)
* Integração com banco de dados em nuvem (Firebase)
* Estruturação de aplicações Java
* Organização e boas práticas de código

## 🛠️ Tecnologias utilizadas

* Java
* Firebase Realtime Database / Firestore
* NetBeans (ou outra IDE utilizada)

## ⚙️ Funcionalidades

* ✅ Cadastro de livros
* 📖 Listagem de registros
* ✏️ Edição de dados
* ❌ Exclusão de registros
* 🔄 Integração em tempo real com Firebase

## 🔧 Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/silvabgui/desafio-CrudJavaFirebase.git
```

2. Abra o projeto na sua IDE (NetBeans, IntelliJ, etc)

3. Configure o Firebase:

* Adicione suas credenciais (google-services.json ou equivalente)
* Configure a conexão no código

4. Execute a aplicação

## ⚠️ Observações

Este projeto utiliza Firebase, portanto é necessário configurar suas próprias credenciais para execução.
